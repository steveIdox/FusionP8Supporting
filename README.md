# FusionP8Supporting

A collection of C# libraries, integrations, and services built in response to customer requests. This repository is intended for developers and implementors who require targeted extensions or services for their solutions.

## Overview

FusionP8Supporting serves as a hub for various supporting code assets including:

- **Libraries** addressing specific service or integration needs described within the repository.
- **Integrations** providing FusionP8 custom lifecycles for streamlined process management.
- **Services** such as the "Signing" WebAPI, featuring an interactive Swagger interface.

> **Note:**  
> This repository houses a mix of .NET versions:  
> - .NET Standard 2.0  
> - .NET Framework 4.7  
> - .NET 8

## Installation

Install via NuGet:

```
dotnet add package FusionP8Supporting
```
or
```
Install-Package FusionP8Supporting
```

[View on NuGet.org](https://www.nuget.org/packages/FusionP8Supporting)
## Usage

- Each library or integration addresses a particular requirement as described in its respective folder or documentation.
- The **Signing** service is exposed as a WebAPI including a built-in Swagger interface for easy interaction and testing.
- Integration modules typically implement FusionP8 custom lifecycles. Refer to each module for implementation details.

## Requirements
